KALI LINUX

Kali é uma distribuição Linux baseada em Debian. O sistema operacional é voltado para auditoria e segurança de computadores. Foi lançado em 13 de março de 2013 pela empresa Offensive Security.

Vejamos a seguir algumas ferramentas do sistema usadas por seguranças da cibernética:

NMAP: Nmap ou “Network Mapper” é uma das ferramentas mais populares no Kali Linux para coleta de informações. Em outras palavras, para obter informações sobre o host, seu endereço IP, detecção de sistema operacional e detalhes de segurança de rede semelhantes (como o número de portas abertas e o que são). Ele também oferece recursos para evasão e falsificação de firewall.

AIRCRACK-NG: Aircrack-ng é uma coleção de ferramentas para avaliar a segurança da rede sem fio. Não se limita apenas a monitorar e obter insights, mas também inclui a capacidade de comprometer uma rede (WEP, WPA 1 e WPA 2). Se você esqueceu a senha de sua própria rede Wi-Fi - você pode tentar usar isso para recuperar o acesso. Ele também inclui uma variedade de ataques sem fio com os quais você pode direcionar / monitorar uma rede Wi-Fi para aumentar sua segurança.

HYDRA: Se você está procurando uma ferramenta interessante para quebrar pares de login / senha, Hydra será uma das melhores ferramentas do Kali Linux que vem pré-instalada.

METASPLOIT FRAMEWORK: Metsploit Framework é a estrutura de teste de penetração mais usada. Ele oferece duas edições - uma (código aberto) e a segunda é a versão profissional. Com essa ferramenta, você pode verificar vulnerabilidades, testar exploits conhecidos e realizar uma avaliação de segurança completa.

MALTEGO: Maltego é uma ferramenta de mineração de dados impressionante para analisar informações online e conectar os pontos (se houver). De acordo com as informações, ele cria um gráfico direcionado para ajudar a analisar a ligação entre esses dados.

BURP SUITE SCANNER: Burp Suite Scanner é uma ferramenta fantástica de análise de segurança na web. Ao contrário de outro scanner de segurança de aplicativo da web, o Burp oferece uma GUI e algumas ferramentas avançadas.

JOHN THE RIPPER: John the Ripper é uma ferramenta popular para cracker de senhas, disponível no Kali Linux. É gratuito e de código aberto também. Mas, se você não estiver interessado na versão aprimorada pela comunidade, você pode escolher a versão profissional para uso comercial.

AUTOPSY FORENSIC BROWSER: A autópsia é uma ferramenta digital forense para investigar o que aconteceu no seu computador. Bem, você também pode usá-lo para recuperar imagens do cartão SD. Também está sendo usado por policiais. Você pode ler a documentação para explorar o que pode fazer com ela.

KING PHISHER: Os ataques de phishing são muito comuns hoje em dia. E a ferramenta King Phisher ajuda a testar e promover a conscientização do usuário, simulando ataques de phishing do mundo real. Por motivos óbvios, você precisará de permissão para simulá-lo em um conteúdo de servidor de uma organização.

NIKTO: Nikto é um poderoso scanner de servidor web - o que o torna uma das melhores ferramentas Kali Linux disponíveis. Ele verifica arquivos / programas potencialmente perigosos, versões desatualizadas de servidor e muito mais.

YERSINIA: Yersinia é uma estrutura interessante para realizar ataques da Camada 2 (a Camada 2 se refere à camada de enlace de dados do modelo OSI) em uma rede. Claro, se você quiser que uma rede seja segura, terá que considerar todas as sete camadas. No entanto, essa ferramenta se concentra na Camada 2 e em uma variedade de protocolos de rede que incluem STP, CDP, DTP e assim por diante.

Você pode instalar o Kali tanto no computador como em uma Virtual Machine.
