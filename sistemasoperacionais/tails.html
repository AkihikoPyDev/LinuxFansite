TAILS LINUX

Tails é uma distribuição Linux baseada em Debian. Tem como objetivo manter a privacidade, segurança e anonimato do usuário, também pretende ajudar a burlar a censura de alguns países. Foi lançado 23 de junho de 2009 pelo Tor.

Tails não é indicado a ser usado em Virtual Machines, pois muitas ferramentas de segurança perdem o "efeito". Você pode instalar o Tails usando um pen-drive de 8gb desligando o computador para iniciá-lo. Tails nunca usa o disco rígido e é independente do outro sistema operacional. O SO pode não te protejer se o computador já tiver vírus ou hardwares maliciosos.

AMNÉSIA: Tails não guarda o que você pesquisa quando desliga o PC. Os Websites, arquivos, senhas e dispositivos Wi-Fi somem. O Tails roda somente na memória do computador.

ARMAZENAMENTO CRIPTOGRAFADO: É possível salver alguns arquivos num Armazenamento Persistente no pen-drive USB: documentos, favoritos do navegador, emails e até alguns programas.

CAIXA DE FERRMANTAS PARA SEGURANÇA DIGITAL: O Tails vem com alguns aplicativos para navegar seguramente, como o TorBrowser com ad-blocker, Thunderbird para emails criptografados, KeePassXC para criar e armazenar senhas fortes, LibreOffice, OnnionShare para compartilhar arquivos usando a rede Tor, etc.

TOR: A rede Tor criptografa e anonimiza a conexão fazendo-a passar por 3 retransmissores, que são servidores operados por diferentes pessoas e organizações espalhadas pelo mundo. O primeiro retransmissor sabe de onde veio, mas não para onde vai; o terceiro sabe para onde vai, mas não de onde veio. O Tor tem mais de 6000 retransmissores.

ANTI-CENSURA: Tor evita que alguém esteja observando a sua conexão saiba o que você está fazendo. Em países como China ou Irá que bloqueia certos sites, é possível usar pontes para se conectar ao Tor.

ANTI-RASTREAMENTO: Tor evita que os sites visitados saibam onde você está e quem é você. É possível também criar uma nova indentidade a partir do Tails.

SOFTWARE ABERTO: O código é aberto, assim, qualquer um pode verificar se o Tails realmente funciona como deveria.

GRATUITO: Para os desenvolvedores, ninguém deveria pagar para para ter segurança ao usar um computador. É por isso que o Tails é de graça e é feito o possível para todos poderem usar. O trabalho é pago via doações para os desenvolvedores.

FUNDAÇÕES SÓLIDAS: Tails é construído sobre fundações sólidas: a rede Tor, o sistema Debian, o ambiente GNOME e todas as ferramentas que vêm no Tails.

Se você quer privacidade, Tails é a melhor solução atualmente!
